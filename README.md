# RTI-CONNECT

RTI-CONNECT is a MERN Stack web application designed to help users access RTI (Right to Information) related services and information efficiently.

## 🚀 Features

- User Authentication
- RTI Information Access
- Responsive UI
- MERN Stack Architecture
- API Integration
- Modern Frontend Design

## 🛠️ Technologies Used

### Frontend
- React.js
- CSS / Tailwind

### Backend
- Node.js
- Express.js

### Database
- MongoDB Compass

## 📂 Project Structure

```bash
RTI-CONNECT/
├── frontend/
├── backend/
└── README.md
```

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/saisreeadepu/rti-connect.git
```

### Install Frontend Dependencies

```bash
cd frontend
npm install
```

### Install Backend Dependencies

```bash
cd backend
npm install
```

## ▶️ Run Frontend

```bash
npm run dev
```

## ▶️ Run Backend

```bash
npm start
```

## 🌐 Deployment

- Frontend: Vercel
- Backend: Render
- Database: MongoDB Atlas
